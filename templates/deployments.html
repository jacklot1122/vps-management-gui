How Local Deploy Works

Local deployments let you push files directly from your laptop to your VPS without using GitHub. This is perfect for projects with secrets, API keys, or config files you don't want on GitHub.

  1. Configure a local deployment below
  2. Copy the generated deploy command
  3. Run it from your laptop whenever you want to deploy
